# 🏭 Industrial Knowledge Intelligence

## Unified Asset & Operations Brain

An AI-powered industrial knowledge assistant that ingests manuals, SOPs, reports, and operational documents to provide intelligent document search, summarization, and question answering.

## Features

- PDF Upload & Processing
- Document Chunking
- Semantic Search using FAISS
- Retrieval-Augmented Generation (RAG)
- Gemini-powered Question Answering
- Source-backed Responses
- Document Summarization

## Tech Stack

- Python
- Streamlit
- Gemini API
- LangChain
- FAISS
- HuggingFace Embeddings
- PyPDF

## Workflow

Upload PDF → Extract Text → Chunking → Embeddings → FAISS Retrieval → Gemini Response

## Future Enhancements

- Knowledge Graph Visualization
- Compliance Checking
- Asset Relationship Mapping
- Multi-document Support
